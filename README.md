# Planet System
### Модель планетарной системы
 
1 курс, 2 семестр, 3 задание, 8 вариант

## Требования к минимальному работоспособному продукту
Разработать программу с графическим интерфейсом, позволяющую задать начальную конфигурацию 
и параметры объектов планетарной системы и позволяющую отобразить
состояние системы через заданное время. Реализовать режим анимации 
движения планет с заданным масштабом времени, а также возможность 
паузы и перемотки на заданный временной интервал. Планеты в системе
вращаются по эллиптическим орбитам, а в одном из фокусов которой 
находится звезда.
